## Legende

| Symbol | Bedeutung |
|--------|-----------|
| `A => B` | Vorwärts — A wird gezeigt, B aufgedeckt |
| `A <= B` | Rückwärts — B wird gezeigt, A aufgedeckt |
| `A <> B` | Bidirektional — zufällige Richtung |
